Step 1: Initiate a nodejs server.
    cmd: npm Init 

Step 2: Install some important dependencies.
    cmd: npm install mysql express nodemon dotenv cors body-parser ejs

Step 3: Create a server.js file.

Step 4: Run the first API.
    cmd: npm start  //for nodemon

Step 5: Create database on localhost (like XAMPP)
Step 6: Connect to the database using the ./config/db.js file.
Step 7: Create .env file
Step 8: Create Pages (e.g create.html)
Step 9: Create API for the pages
